defininition of terms


inline css
this is done by adding the style attribute to the html elements.no selectors needed

internal css
this is where the styling is done in the head section of the web. uses the selector and the declation

external CSS
the styling is done in a different file and then linked to the html web page .uses the selectors and declaration block
